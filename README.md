# Sport Center

**Sport Center** is a modern booking system that simplifies sports court reservations with **real-time availability, instant confirmations, and an intuitive user interface**. Built to make court bookings effortless for users while reducing administrative overhead.

## Key Contributions
- Designed and implemented a **real-time booking calendar** to prevent double bookings.  
- Streamlined reservation workflow, enhancing user convenience and reducing manual errors.  
- Developed **interactive UI** for easy selection of courts and time slots.  
- Built backend logic for **automatic booking confirmations and notifications**.
